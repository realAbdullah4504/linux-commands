## Quick Reference

### Essential Commands
```bash
# Initialize repository
git init

# Add files to staging
git add .

# Commit changes
git commit -m "message"

# Push to remote
git push origin main

# Pull latest changes
git pull origin main

# Check status
git status
```

### Branching Strategy
- `feature/branch-name` - New features
- `bugfix/branch-name` - Bug fixes
- `hotfix/branch-name` - Urgent production fixes
- `release/branch-name` - Releases and versioning
- `chore/branch-name` - Minor changes like docs
